# Как внедрять свои функции:
в файле **calc.py** пишете необходимую функцию
после чего в файле **main.py** вы можете добавить свою функцию в словарь
```python
self.commands = {"+-*/":Calc.func}
```
указывая оператор и саму функцию

так же в этом же файле добавляете свою кнопку в функцию
```python
def create_buttons(self):
```
по примеру 
```python
self.addition_button = ttk.Button(self.root, text="+", command=lambda: self.button_handler(self.addition_button["text"]))
self.addition_button.grid(column=1, row=3, sticky="ns")
```

вы можете менять поведение приложения на свои кнопки в функции
```python
def button_handler(self, command_type):
```

В принцепе интерфейс мы можем поменять если предложите свой вариант.
